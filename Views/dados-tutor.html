Dicas Importantes

Utilize esta opção para os seus dados pessoais, assim como a sua senha quando achar necessário;

Caso não queira alterar a sua senha, basta deixar os campos referentes à senha vazios;

Essas informações são muito importante, para que o pessoal da unidade de saúde possa localizá-lo com facilidade quando for necessário.


Para atualizar seus dados faça:

Estando na opção [Meus Dados];

Digite os dados preenchendo os campos com atenção especial aos campos obrigatórios assinalados com um (*);

Clicar no botão [Salvar].


NOTA: O cadastro de tutores está restrito ao município de São Luís-MA.
